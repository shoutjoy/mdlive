# 상단 1~500줄 기준 export 가능 블록 (단순 텍스트 기준, 해석/리팩터링 없음)

- 줄 1~3: ins, repCL (함수 2개) — 유지 주석 있음
- 줄 20~219: const IPPT = (() => { ... })();  — 한 블록
- 줄 234~313: const A4Ruler = (() => { ... })(); — 한 블록
- 줄 322~331: CP 관련 시작 (이후 다른 블록과 섞여 있을 수 있음, 500줄 밖)
- 줄 339~532: const SS = (() => { ... })(); — 한 블록

export 시: 위 줄 구간을 그대로 잘라서 별도 파일로 두고, 원본에서는 해당 구간 삭제 후 한 줄 주석으로 대체하는 방식만 적용.
